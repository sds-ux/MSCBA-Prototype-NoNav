Processus serveur : Confirmer le propriétaire avec l'ARC


When a user claims to be the business owner, the system checks if this is true using the following steps:

  1. Extracting the BN-9 from the BN-15 in found in the BUS database.
  2. Using the BN-9 to retrieve the business owner’s name from the BN Validation Service – Note: CRA data can contain up to five business owners per BN.
  3. Comparing the name submitted by the user with the business owner names in the BN Validation service.

To continue, the first name and last name must be a perfect match.


If the user is confirmed as the owner, continue with owner consent process.

Continuer


If the user is not the owner, the system displays an error message then closes the secure session.
