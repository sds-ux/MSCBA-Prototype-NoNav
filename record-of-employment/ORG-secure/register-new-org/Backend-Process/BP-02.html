Backend process: Confirm user’s identity


Once the user has logged in their CRA My Account through Portageur to attempt to confirm their identity, Portageur redirects the user back to MSCBA and returns to MSCBA the data that the user agreed to share.

The system compares the user’s first and last name from their profile with the first and last name received from the CRA data.

To continue, the first name and last name must be a perfect match


If the names are a match, the identity of the user is confirmed and they can continue with the business owner consent process.

Continue


If CRA data fails to confirm the user’s identity, the system generates a work item called “Online Identification” and redirects the flow away from owner consent.
