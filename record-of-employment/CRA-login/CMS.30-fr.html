Avez-vous oublié votre ID utilisateur?

Vous ne pouvez pas récupérer votre ID utilisateur si vous l''avez oublié ou égaré.

Vous devez vous inscrire à nouveau pour Mon dossier, Mon dossier d'entreprise, Représenter un client, MonARC, Livraison de données fiscales, Carrières, ou aller à la page d'accueil de l'ARC.

Gérer mes options de sécurité de l'ARC

Pour gérer vos options de sécurité, par exemple pour révoquer votre ID utilisateur de l'ARC, modifier votre mot de passe, gérer vos questions et réponses, etc., vous devez d'abord ouvrir une session à un service. Une fois que vous avez ouvert une session, la page « Dernière ouverture de session à l'ARC » vous sera présentée. Sur cette page, vous trouverez des liens pour gérer vos options de sécurité de l'ARC.
